A2Go: Ann Arbor-themed PokemonGo clone

This project was inspired by PokemonGo and is a mobile app that incorporates extended-reality elements. Ann Arbor claims that it is the Tree Town, so instead of collecting Pokemon you collect seeds and then use them to plant trees. Periodically squirrels will spawn next to the trees, preventing them from growing. In this case you must switch to interaction mode to feed the squirrels an acorn. You can also change the color of you avatar if you're sick of the bland color pallete. This project was made using Unity (2022) with the MapBox plugin. We didn't create any of the components or animations ourselves, but found them online using free resources such as Sketchfab and Mixamo
